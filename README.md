# 🚨 CrisisLink AI

An AI-powered emergency response and crisis management system designed to optimize real-time decision-making during critical situations.

---

## 🌍 Overview

CrisisLink AI transforms how cities respond to emergencies by combining:

- 🧠 AI-driven incident analysis (Google Gemini)
- 🗺️ Real-time routing and traffic optimization (OSRM + OpenStreetMap)
- 🚦 Intelligent signal corridor activation
- 🚗 Live traffic simulation with moving vehicles
- 📊 Command center dashboard for real-time monitoring

The system enables faster response times, better coordination, and smarter resource allocation in high-pressure scenarios.

---

## ⚡ Key Features

- 🤖 Natural language emergency reporting using AI
- 🚑 Automated dispatch of responders
- 🛣️ Smart route optimization with congestion avoidance
- 🚦 Dynamic green corridor system
- 📡 Live command feed with system transparency
- 🎮 Simulation mode for multi-incident stress testing
